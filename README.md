# Authentication App

Simple app to demonstrate authentication using React for the frontend framework and making calls to a Express.js framework Node.js backend.

## Technologies
- Frontend
   - React
   - axios
   - react-icons
   - react-router-dom
   - react-toastify

- Backend
   - Node.js
   - Express.js
   - MongoDB
   - mongoose
   - dotenv
